# Direction

## Claim

The same four-stage support path can answer two different questions without changing its semantic layout: a focus marker shows where active attention is now, while a forwarding arrow shows who hands ownership to whom.

## Relation

Sequence / process. The stage order is fixed in both variants; only the meaning encoded by motion changes.

## Audience

Service operations teams comparing two stakeholder readings of one shared support process.

## Composition

A single left-to-right timeline with four equal stage cards, one owner badge per stage, and three fixed connectors. Both variants use the same canvas, coordinates, labels, colors, and static structure so the comparison isolates motion semantics.

## Animation Stories

- **Dot / active focus:** exactly one stage receives the active-focus pulse at a time, advancing in process order and wrapping to intake. This follows the Focus / emphasis rule: the pulse means “attention is here now.”
- **Arrow / ownership handoff:** one arrow advances along each connector in process order, then the next transfer begins. This follows the Sequence / process rule: direction stays forward and motion literally travels from the current owner toward the next owner.

Each state holds for two seconds in an eight-second seamless loop. The poster timestamp is chosen late enough that the complete static process is visible; motion adds emphasis but carries no missing stage or owner information.
