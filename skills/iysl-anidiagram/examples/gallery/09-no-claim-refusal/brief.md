# Team Tooling Reference Brief

A team maintains an internal wiki page listing the tools each squad uses:

- Squad Atlas: issue tracker, shared drive, weekly digest bot.
- Squad Beacon: issue tracker, design library, incident pager.
- Squad Cinder: shared drive, incident pager, release dashboard.

The page also records each tool's owner and renewal month. Nothing on the page argues that one squad's setup is better, that tooling is converging or diverging, or that any change is needed. It exists so people can look up who owns what.
